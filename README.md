# Sorting Algorithm Visualizer

A web-based Sorting Algorithm Visualizer that helps users understand how different sorting algorithms work through animated visual representations. It displays array elements as bars and shows the sorting process step by step.

## 🚀 Features

- Generate a random array
- Visualize sorting step by step
- Adjustable sorting speed
- Adjustable array size
- Interactive and user-friendly interface
- Real-time comparison and swapping animations
- Supports multiple sorting algorithms

## 🧠 Sorting Algorithms Included

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
